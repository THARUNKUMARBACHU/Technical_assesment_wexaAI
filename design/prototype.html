Dashboards

Org Switcher Flow
1. User clicks the org area at bottom of sidebar
2. Dropdown appears showing all organizations the user belongs to
3. Current org is highlighted with checkmark and blue background
4. Each org shows: name, user's role, member count
5. Clicking another org: calls POST /auth/switch-org, gets new JWT, page reloads with new org's data
6. "Create new organization" opens the Create Org form
7. The user's role can differ per org (Owner in ShopCo, Analyst in StartupXYZ)

What changes when switching orgs:
